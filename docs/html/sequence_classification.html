class CustomClassifier(name='CustomName', transformer=None)[source]¶

Bases: sequence_classification.classifiers.sequence_classifier.SequenceClassifier

_fit(X, y)[source]¶

Fit method is responsible mainly for training model.

It should also check the parameters. New attributes can be also added here to the object - they should be ended by _ at the end, e.g. self.fitted_. Fit method should return self (for compatibility reasons with common interface of scikit-learn). Source: http://danielhnyk.cz/creating-your-own-estimator-scikit-learn/

X : features to train on y: vector of classes to train on, same length as X

self: the classifier itself

_predict(X)[source]¶

Predict method is responsible for predicting the results based on X.

It should return predictions.

X : features to predict from

y: vector of predicted classes, same length as X
